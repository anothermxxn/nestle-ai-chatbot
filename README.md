# Nestlé AI Chatbot

An AI-based chatbot for the Made with Nestlé website that can handle user queries, generate appropriate responses, and provide reference links where necessary.

## Tech Stack

### Backend
- FastAPI (Python)
- Azure OpenAI
- Azure Cognitive Search

### Frontend
- Vite + ReactJS

## Prerequisites
- Python 3.11
- Node.js 20+
